## 概要

このアプリは、プログラミング言語やライブラリ別に学んだことを、一問一答形式でメモしておき、単語カードのように覚えているかを確認できるアプリです。

ログインなしでもご利用いただけますので、お忙しい場合はそのまま閲覧ください。

[アプリのリンク]('https://anki-note.web.app')
https://anki-note.web.app
